cito.Database.DBBase module¶

Base class for all DB interfaces.

This class provides various functions for creating connections to MongoDB and also locating the MongoDB server.

cito.Database.InputDBInterface.get_db_connection(hostname='127.0.0.1')[source]¶

Get database connection objects for the input or output databases.

This function creates MongoDB connections to either the input or output databases. It also maintains a cache of connections that have already been created to speed things up.

Parameters:
  • hostname (str.) – The IP or DNS name where MongoDB is hosted on port 27017
  • selection (str.) – ‘input’ for EventBuilder input, blocks, otherwise ‘output’ for output
Returns:

list – [pymongo.Connection, pymongo.Database, pymongo.Collection]
Raises:

pymongo.errors.PyMongoError
